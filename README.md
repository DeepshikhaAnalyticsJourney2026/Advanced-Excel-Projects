# Excel Practice Assignments
## Description
This repository contains Excel practice assignments created to demonstrate practical knowledge of formulas, calculations, and basic data analysis.  
The project includes real-world examples such as student results, employee salary sheets, sales reports, and inventory data.
## File Details
- File: my assignment.xlsx  
- Tool: Microsoft Excel  
- Sheets: 11  
## Key Assignments
- Student Marks & Grade Calculation  
- Item Billing & Cost Analysis  
- Employee Salary & Payroll Sheet  
- Monthly & Seasonal Sales Reports  
- Inventory & Order Data Analysis  
## Excel Skills Used
- SUM, AVERAGE  
- IF Formula  
- Percentage Calculation  
- Date Formatting  
- Data Organization  
## Learning Outcome
- Practical Excel experience  
- Understanding of business datasets  
- Improved accuracy and reporting skills  
- Strong base for Data Analysis & Power BI
